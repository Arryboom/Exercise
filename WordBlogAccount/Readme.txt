管理 Word 博客账户的 VC++ 程序

1、运行：bin\WordBlogAccount.exe
2、操作：doc\管理 Word 博客账户-16.11.05.doc
3、编译：
   以 VC++2015 为例。请打开下面两个文件之一
   makeATL\vc2015\WordBlogAccount.sln     使用 ATL 构建程序界面
   makeMFC\vc2015\WordBlogAccount.sln     使用 MFC 构建程序界面
   配置：
   DA——Debug Ansi 版
   DU——Debug Unicode 版
   RA——Release Ansi 版
   RU——Release Unicode 版
   bin\WordBlogAccount.exe 是由 makeATL\vc6\WordBlogAccount.dsw 编译出来的 RA 版本
